Ytravel
========

A mashup app of Flicker + Foursquare + Google maps + Yahoo weather which will help you explore while travelling between two cities

## Components

### Frontend

* EJS
* Zurb Foundation CSS Framework
* JQuery

### Backend

* Node.js
* Express.js
* Mongoose.js

### Database

* Mongo
